# Raildock Frontend

## 프로젝트 소개
Raildock 서비스의 프론트엔드 프로젝트입니다.  
사용자에게 철도 유지보수 시스템을 제공합니다.
선로/애자/둥지 탐지 기능을 제공하여 상태기반 유지보수에 도움이 됩니다

## 개발환경
- React
- TypeScript
- Vite
- Tailwind CSS
- ShadCN
- React Query
- Axios

## 실행 환경
### 요구사항
- Node.js 20+
- npm 9+

### 환경 변수
프로젝트 루트 디렉토리에 .env 파일 생성

아래 내용 추가
```
VITE_BACKEND_URL= 백엔드 URL 
```

### 패키지 설치 및 실행
```
npm install
npm run dev
```
